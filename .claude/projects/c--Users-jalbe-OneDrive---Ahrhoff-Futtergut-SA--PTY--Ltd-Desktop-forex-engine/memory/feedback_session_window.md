Tested 16 session window combinations across 4 pairs on 50k bars. 12-16 UTC won decisively.

**Why:** Wider windows (earlier start or later end) let in signals from lower-liquidity hours that have worse win rates. The London/NY overlap (12-16 UTC) has the tightest spreads and most directional moves.

**How to apply:** Don't revisit session windows unless changing pairs or timeframe. Per-pair windows (e.g., Tokyo for GBPJPY) also tested implicitly — didn't help. Keep 12-16 UTC.
